# TETRIS
## 特徴
### テトリスガイドラインに準拠した仕様
- スーパーローテーション
- バッグシステム
- ロックダウン

### No GameEngine
- DxLibを使用して作成

## ビルド
### 前提
- MinGWを使ってgccのセットアップをしてください
  - gccのバージョンは3.10が好ましいです

### `make`の実行
```
make
```


## 開発memo
- LLVMをセットアップしていると，リンターも動くので開発がはかどります
